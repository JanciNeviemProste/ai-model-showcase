Závery

Kľúčové zistenia

🏆

Celkový víťaz

Claude Opus 4.6

Najrýchlejší (10m 47s), funguje okamžite bez opráv. Carousel, mapa, animované čítače, focus trap. Vanilla HTML/CSS/JS = žiadne build kroky, žiadne závislosti.

👻

Neviditeľný web

Claude Sonnet 4.6

Excelentný kód (TypeScript strict, React hooks), ale výsledok je kompletne neviditeľný. 45 elementov s opacity:0 od Framer Motion. Na statickom hostingu = biela stránka.

⚡

Spoľahlivá dvojka

Kimi 2.5

Funguje okamžite bez opráv. Canvas particle animácia, fluid typografia, vanilla JS. Žiadne závislosti. Druhý najlepší funkčný výsledok.

💀

Nedokončený projekt

Gemini 3.1 Pro

Homepage = default Next.js "To get started, edit page.tsx". Najpomalší (23m 31s), chaotická štruktúra, a ešte sa aj pýtal otázky. Komponenty existujú, ale projekt je rozbitý.

Víťazi v kategóriách

💡 Hlavné zistenie

Fungujúci web > dokonalý kód. Opus a Kimi použili vanilla HTML/CSS/JS a ich weby fungujú okamžite. Sonnet a Gemini zvolili React+Next.js stack — Sonnet odovzdal neviditeľný web (Framer Motion opacity bug), Gemini nedokončil ani homepage. Prepni na "Ako dodané" aby si videl realitu.

React nie je vždy odpoveď. Pre prezentačný web vanilla stack fungoval spoľahlivejšie. Oba vanilla modely (Opus, Kimi) odovzdali funkčný produkt. Oba React modely (Sonnet, Gemini) vyžadovali manuálne opravy.
